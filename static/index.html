Project DeltaX

Project DeltaX is a project meant to demonstrate the power of Rust via Axum, and later on, integration with other libraries like Leptos.

In the future, this website would have been created via WASM or Web Assembly.

For now, it's a static HTML page.

It also features multiple libraries like Tokio stack, containing:

  1. Hyper
  2. axum
  3. Tower
  4. Tracing
